HyPrism Flatpak Remote

This is the experimental Flatpak remote for HyPrism. Use it if you want automatic updates via Flatpak. If something breaks, use the regular GitHub Releases instead.

Quick install

Add the remote:

flatpak remote-add --if-not-exists hyprism https://xargonwan.github.io/HyPrism/flatpak-repo

Install HyPrism:

flatpak install hyprism dev.hyprism.HyPrism

Update later:

flatpak update dev.hyprism.HyPrism

Notes

  • This remote is experimental and may be updated or replaced.
  • For older Linux systems, Flatpak may work better than native builds.
  • Releases are hosted on GitHub: HyPrism releases.
