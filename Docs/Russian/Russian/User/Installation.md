# Установка

## Требования

- **Windows:** Windows 10 **22H2** или Windows 11 (x64)
- **Linux:** x64-дистрибутив, совместимый с современными сборками Electron (базово проверяется: Ubuntu 18.04+, Debian 10+, Fedora 32+)
- **macOS:** Monterey (12)+, Apple Silicon (arm64)
- Интернет-соединение для первого запуска (аутентификация и загрузка игры)

Примечание по рантайму:
- Текущие релизные сборки HyPrism используют **Electron 34.2.0** и self-contained публикацию `net10.0`.
- Отдельная установка .NET Runtime для релизных бинарников не требуется.
- На старых сборках Windows 10 (ниже 22H2) приложение может не запускаться из-за неподдерживаемой комбинации OS/рантайма.

## Загрузка

Скачайте последний релиз со страницы [GitHub Releases](https://github.com/yyyumeniku/HyPrism/releases).

### Windows

Вариант A (portable):
1. Скачайте `HyPrism-win-x64.zip`
2. Распакуйте в любую папку
3. Запустите `HyPrism.exe`

Вариант B (установщик):
1. Скачайте `HyPrism-win-x64-<version>.exe`
2. Запустите установщик
3. Запустите HyPrism через меню Пуск или ярлык на рабочем столе

### Linux

HyPrism использует нативные Linux-утилиты диалогов для выбора папок и файлов. Установите хотя бы одну из них:
- `zenity`
- `kdialog`
- `yad`
- `qarma`

#### AppImage
1. Скачайте `HyPrism-linux-x64.AppImage`
2. Сделайте файл исполняемым: `chmod +x HyPrism-linux-x64.AppImage`
3. Запустите: `./HyPrism-linux-x64.AppImage`

#### Flatpak
```bash
flatpak install io.github.HyPrismTeam.HyPrism
flatpak run io.github.HyPrismTeam.HyPrism
```

### macOS

1. Скачайте `HyPrism-osx-x64.zip` (или `osx-arm64` для Apple Silicon)
2. Распакуйте и переместите `HyPrism.app` в Программы
3. Запустите из папки Программы

## Каталог данных

HyPrism хранит свои данные (конфигурация, экземпляры, профили, логи) в:

| ОС | Путь |
|----|------|
| Windows | `%APPDATA%/HyPrism/` |
| Linux | `~/.local/share/HyPrism/` |
| macOS | `~/Library/Application Support/HyPrism/` |

### Структура каталога

```
HyPrism/
├── config.json         # Конфигурация лаунчера
├── Instances/          # Установки игры (папки на основе GUID)
│   └── {GUID}/         # Отдельный экземпляр
├── Profiles/           # Профили игроков и резервные копии скинов
├── Logs/               # Логи приложения
└── Cache/              # Временные файлы
```

## Первый запуск (начальная настройка)

При первом запуске HyPrism проведёт вас через мастер начальной настройки:

1. **Экран приветствия** — Добро пожаловать в HyPrism
2. **Выбор языка** — Выберите предпочитаемый язык (12 доступно)
3. **Аутентификация Hytale** — Войдите в свой аккаунт Hytale
4. **Настройка профиля** — Создайте первый профиль игрока (никнейм, аватар)
5. **Начальные настройки** — Настройте предпочтение GPU и другие параметры

После начальной настройки:
- Лаунчер создаёт структуру каталога данных
- Ваш профиль и настройки сохраняются
- Вы можете загрузить и установить игру с панели управления
