# Contributing to HyPrism

Thanks for contributing! Please follow these guidelines:

- Fork the repo and create a branch `feature/your-feature` or `fix/your-bug`.
- Add tests when applicable.
- Follow code style conventions for C# and TypeScript.
- Open a Pull Request with a concise description and link to any related issues.

Release flow

- Releases are managed by GitHub Actions (see `.github/workflows/release.yml`).
- Packaging for Linux uses the scripts in `scripts/` and `packaging/flatpak/`.
